# Backend System Requirements & Dependencies

While this is a Node.js project (managed via package.json), this file outlines the core system requirements and high-level module dependencies required to run the backend API.

## System Requirements
- Node.js >= 18.0.0
- Docker Desktop (or Docker Engine + Docker Compose)
- PostgreSQL (Provided automatically via docker-compose)

## Core Runtime Dependencies (NPM)
express==4.18.2
@prisma/client==5.10.2

## Development Dependencies
